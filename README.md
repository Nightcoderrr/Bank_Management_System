# Bank Management System

A command-line based **Real-World Bank Management System** 
built using **Python** and **Object-Oriented Programming (OOP)** concepts.

# Features
- Create & manage bank accounts
- Deposit & Withdraw money
- Transfer money between accounts
- View transaction history
- Password protection per account
- Admin view for all accounts
- Change password & delete account

# OOP Concepts Used
- Classes & Objects
- Constructors (`__init__`)
- Instance methods & attributes
- Encapsulation

#Tech Stack
- Language : Python 3
- Database : No external DB (in-memory)
- Libraries : None (pure Python)

#How to Run
```bash
python bank_management.py
```

# Default Flow
1. Create a new account
2. Login with account number & password
3. Perform transactions
